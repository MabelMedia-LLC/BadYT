# BadYT
BadYT - A List Of Youtube Channels That Violate Youtube's TOS.

# Found Any Not On This List?
Fork The Repo, Add Them To The List, And Open A Pull Request, And We'll Get To Reviewing The Urls Right Away.

Please Provide Evidence That The Channel In Question Has Done What You Say They Have Done (Screenshots, Urls).

Please Format The PR Like So:
Channel Url | Channel Name | TOS Term Violated | BotNet ID (If Part Of A BotNet).

# Achievements 🥇 🏆
When A Channel On The List Is Terminated, We'll Move It To `Terminated.txt`.

# Why Are My Channel(s) On This List? :rage:
If You Suspect Your Channel Was Added To This List In Error, Please Open A Github Issue, With The Entries That Specify Your Channel(s).

If Someone Has An Open PR Adding Your Channel(s), Feel Free To (In A Civil Manner) Discuss With The Submitter In It's Comments.
Any Harrasment Of Submitters / Channel Owners Will Not Be Permitted And All Such Conversations Will Be Locked.
